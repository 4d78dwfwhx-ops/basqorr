## عن المشروع

**بازقر** منصة رقمية سعودية توحد كل ما يتعلق بالمركبات في مكان واحد:
- سوق السيارات (جديد، مستعمل، مزادات)
- سوق قطع الغيار (بحث ذكي برقم الهيكل)
- خدمات المساعدة على الطريق (SOS)
- حجز الورش والخدمات
- نظام أعمال متكامل (فواتير، ضريبة)

---

##  المشكلة والحل

| المشكلة | حل بازقر |
|---------|----------|
| تشتت منصات السيارات والخدمات | منصة واحدة شاملة |
| صعوبة العثور على قطع الغيار | بحث برقم الهيكل مباشرة |
| ضعف الثقة في المعاملات | دفع آمن وبوابات سعودية |
| غياب سجل موحد للمركبات | سجل وطني طوعي للمالكين |

---

##  الميزة الأساسية: البحث برقم الهيكل (VIN)

أدخل رقم الهيكل فقط، والنظام:
- يفك تشفير الموديل والسنة والسوق
- يطابق قطع الغيار المتوافقة
- يكشف تاريخ الحوادث (إن وجد)
- يحمي خصوصية المالك
